2. 危险报警灯

4. 刹车踏板
5. 方向盘转角
6. 方向盘转角速度
7. yawrate

## 工作条件
1. 速度工作区间：工作范围（speed_min, speed_max）, 下限滞回速度， 上限滞回速度
2. 车道宽度(例如 > 2.8)
3. 曲率半径(例如 > 250)
4. 最大横向车速

    kLdwStandby = 1,  // 待机状态

    kLdwPassive = 3   // 信号抑制状态
